Frequently Asked Questions

Find answers to common questions about Pteams

What is Pteams? +

Pteams is a software solution that allows you to use smartphones, tablets, and other computers as additional monitors for your Windows PC over your local network or the internet. It creates a virtual display that can be accessed from any compatible device.

Is Pteams free to use? +

Yes, the basic version of Pteams is completely free for personal and commercial use. There may be premium features available in the future, but the core functionality will remain free. We believe in providing value to our users without hidden costs.

What are the system requirements? +

For the server: Windows 10 or 11 (64-bit or ARM64). For viewers: Android 5.0+, iOS 12.0+, or any modern web browser for the HTML5 version. All devices need to be connected to the same network for automatic discovery, though manual connection is also possible.

Does it work over Wi-Fi? +

Yes, Pteams works over both Wi-Fi and Ethernet connections. For best performance, we recommend using a 5GHz Wi-Fi network or wired Ethernet. The software automatically adapts to network conditions to provide the optimal balance between performance and quality.

Can I use touch input on secondary displays? +

Yes, Pteams supports touch, pen, and mouse input on compatible secondary devices, effectively turning them into software KVM switches. This means you can interact with applications on your primary PC directly from your secondary devices.

How many devices can I connect simultaneously? +

There's no hard limit on the number of devices you can connect. Performance will depend on your hardware and network capabilities. Most users successfully connect 2-4 additional displays without any issues. The software is optimized to handle multiple connections efficiently.
